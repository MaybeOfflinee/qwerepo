Kurulum

- Dosyaları indirdikten sonra winrardan masaüstüne çıkartın.
- Java versionu 281 veya 291 olması lazım java versionunuzu CMD ye "java -version" yazarak kontrol edebilirsiniz.
- Uygulamayı açın ve register bölümüne tıklayın, burada kullanıcı ismi ve şifre voidin kendi datasına kaydoluyor craftrise ile alakasızdır. License bölümü yerinede siz verilen lisansı yazınız ve registere basınız.
- Başlat butonu geldiğinde kaydınız tamamlanmış demektir ve buraya craftrise hesabınızı ve şifrenizi giriniz RAM bölümüne sırasıyla : 

  1024 -> 1 GB Ram
  2048 -> 2 GB Ram
  4096 -> 4 GB Ram
  
ve katları...

- GIF Size gifli cape özelliğini kullanabilmeniz için eklenmiştir orayı isterseniz boş bırakabilirsiniz aşşağı kısımda gif kurulumu bölümünde ayrıntılı bir şekilde anlatacağım.
- Ardından başlata basıp girişinizi sağlayabilirsiniz.

GIF Yükleme

- İnternetten sevdiğiniz bir gifi alıp launcherın bulunduğu klasöre atınız.
- İlk defa gifi yüklüyorsanız GIF Yükle butonuna basınız eskiden olan bir gifin üstüne yüklüyorsanız önce GIF Sil yapın ardından GIF Yükle yapın.
- GIF Size yerine 12,17 yazın formatı şu şekilde genişlik,uzunluk.

Bilinen Hatalar

- 2 tane java bulundu veya javanız doğrulanamadı hatası 1 den fazla javanız bulunduğunda veya javanızın sürümü uyuşmadığından verir bunu tüm javaları silip https://java.com/ dan yenisini indirerek çözebilirsiniz.
- Özel durum hatası alanlar çalıştırdan "C:\Windows\System32\drivers\etc" bu yola gitsin ve hosts dosyasına sağ tıklayıp özelliklere tıklaın ve güvenlik bölümüne gelip tam denetim veriniz.
- Java Virtual Machine Launcher başlığında hata alanlar RAM'a 1024 verip denesin.
- Bazı bilgisayarlarda oyun tam ekran başlayınca siyah ekran oluyor olan kişiler pencereli başlatsın.
